# pyjnius-install
bash script to install pyjnius and requirements

tested on:
Google Cloud Platform
Debian GNU/Linux 9 (stretch)
December 9, 2017

Installation Method:

  $ git clone https://github.com/chrisseymour/pyjnius-install.git
  
  $ chmod +x inst.sh
  
  $ yes y | ./inst.sh
  
  $ source paths.sh
